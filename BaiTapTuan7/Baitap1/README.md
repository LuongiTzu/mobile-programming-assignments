# UTHSmart - Color Theme App

Ứng dụng Android được phát triển bằng Jetpack Compose để quản lý và hiển thị các theme màu sắc một cách sinh động và trực quan.

## Tính năng chính

- Quản lý theme - Thay đổi màu sắc giao diện động
- Xem chi tiết màu - Hiển thị thông tin màu sắc đẹp mắt
- Modern UI - Giao diện hiện đại với Material Design 3
- Reactive - Cập nhật giao diện ngay lập tức khi theme thay đổi

## Công nghệ sử dụng

- Kotlin - Ngôn ngữ lập trình
- Jetpack Compose - UI Toolkit hiện đại
- Material Design 3 - Hệ thống design
- Navigation Component - Điều hướng giữa các màn hình
- MVVM Architecture - Kiến trúc ứng dụng
- State Management - Quản lý state với Flow/StateFlow
